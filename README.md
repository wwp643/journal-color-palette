# 📊 高分期刊数据配色速查表 (Journal Color Palette)

这是一个专为学术论文和科学数据可视化设计的在线配色速查工具。它收录了多组（2组至10组）高可读性、高专业度的 RGB 配色方案，旨在帮助科研工作者快速选择适合图表和数据展示的颜色组合，提升论文图表的专业度和视觉效果。

## ✨ 核心特色

* **专业严谨：** 所有配色方案均基于对高分学术期刊的图表分析，确保色彩的学术规范性。
* **分组明确：** 提供从 **2组** 到 **10组** 的系统化配色，完美匹配不同实验设计和数据维度。
* **场景匹配：** 每套方案都附带详细的**适配场景**（如“梯度变量”、“冷暖对冲”等）和**文案说明**，指导您在何种图表中使用。
* **一键复制：** 在线网页中，点击配色代码即可实现一键复制，快速导入到 Origin、R、Python 或其他绘图软件中使用。

## 🚀 如何使用

### 1. 访问网站

本配色表已通过 GitHub Pages 部署为可直接访问的网站。

🔗 **访问链接:** `https://wwp643.github.io/journal-color-palette/`


### 2. 在线操作指南

* **筛选:** 点击顶部的“2组”、“3组”等按钮，快速筛选所需数据组数的配色方案。
* **查看:** 每个卡片展示了配色风格、实际色块预览以及适用的场景描述。
* **复制:** 鼠标点击配色方案的 RGB 值区域，即可将全部 RGB 代码一键复制到剪贴板。

### 3. 本地使用 (开发者)

如果您想在本地运行或修改代码，请遵循以下步骤：

1.  克隆仓库到本地：
    ```bash
    git clone [https://github.com/](https://github.com/)[您的用户名]/journal-color-palette.git
    ```
2.  进入项目目录：
    ```bash
    cd journal-color-palette
    ```
3.  使用浏览器打开 `index.html` 文件即可。

## 🎨 配色方案数据结构

所有配色数据都以 JSON 格式存储在 `color_palette.html` 的 JavaScript 代码中。核心字段如下：

| 字段名称 | 说明 | 示例值 |
| :--- | :--- | :--- |
| `groups` | 颜色组数 (用于筛选) | `3` |
| `style` | 配色风格名称 | `学术严谨风` |
| `rgb` | 颜色 RGB 值（由 `+` 或 `→` 分隔） | `246,124,111 + 130,196,182` |
| `scene` | 适配的图表场景描述 | `三种实验处理组的对比分析` |
| `caption` | 详细的配色描述 | `冷暖平衡的严谨配色...` |
